# cbr2cbz
Convert CBR comic book files to CBZ format


#### Dependencies
- p7zip
- p7zip-rar

###### Debian/Ubuntu
```sh
sudo apt install p7zip p7zip-rar
```

###### Arch Linux
```sh
sudo pacman -S p7zip
```


#### Usage
Make executable
```sh
chmod +x cbr2cbz.sh
```

Convert
<pre>
./cbr2cbz.sh <i>source_directory</i> <i>target_directory</i>
</pre>

Help
```sh
./cbr2cbz --help
```
